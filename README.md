# Job Application Tracker

Full stack project built with:

Backend
- ASP.NET Core Web API
- Entity Framework Core
- PostgreSQL

Frontend
- React (Vite)

Features
- Track job applications
- Status management
- Interview scheduling
- Dashboard statistics
- Pagination
- Filters & sorting
- Charts for analytics

Deployment
- Backend: Render
- Frontend: Vercel
- Database: Neon PostgreSQL
